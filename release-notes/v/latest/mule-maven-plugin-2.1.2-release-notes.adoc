= Mule Maven Plugin 2.1.2 Release Notes

MuleSoft is pleased to announce the release of the Mule Maven Plugin 2.1.2. This release fixes a bug where redeployment failed for an existing application in ARM.

== Hardware and software requirements
JRE 1.7.0 (Recommended JRE 1.7.0_79/80), JRE 1.8, Maven 3.3.x

== Compatibilities
* Access Management API 1.0
* ARM Rest API 1.1
* CloudHub API 1.0
* Runtime Manager Agent API 1.0
* Mule Kernel, Mule 3.x
