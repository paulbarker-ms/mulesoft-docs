[[trim1]]
===== trim(String): String

Removes any excess spaces at the start and end of a string.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  "a": trim("   my long text     ")
}
----

.Output
[source,JSON,linenums]
----
{
  "a": "my long text"
}
----
