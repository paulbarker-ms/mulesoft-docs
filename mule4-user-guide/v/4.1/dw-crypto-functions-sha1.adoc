[[sha11]]
===== SHA1(Binary): String

Computes the SHA1 hash and transforms and transforms the binary result into a hexadecimal lower case string.

.Transform
----
%dw 2.0
import dw::Crypto
output application/json
---
Crypto::SHA1("dsasd" as Binary)
----

.Output
----
"2fa183839c954e6366c206367c9be5864e4f4a65"
----
