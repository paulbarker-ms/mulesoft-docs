[[min1]]
===== min(Array<T>): T | Null

Returns the lowest element in an array.
Returns null when the array is empty

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: min([1, 1000]),
  b: min([1, 2, 3]),
  d: min([1.5, 2.5, 3.5])
}
----

.Output
[source,JSON,linenums]
----
{
  "a": 1,
  "b": 1,
  "d": 1.5
}
----
