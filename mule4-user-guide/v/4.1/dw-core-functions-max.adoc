[[max1]]
===== max(Array<T>): T | Null

Returns the highest element in an array.
Returns null when the array is empty

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: max([1, 1000]),
  b: max([1, 2, 3]),
  d: max([1.5, 2.5, 3.5])
}
----

.Output
[source,JSON,linenums]
----
{
  "a": 1000,
  "b": 3,
  "d": 3.5
}
----
