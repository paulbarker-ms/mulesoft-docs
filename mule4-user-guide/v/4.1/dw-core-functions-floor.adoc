[[floor1]]
===== floor(Number): Number

Rounds a number downwards, returning the first full number below
the one provided.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: floor(1.5),
  b: floor(2.2),
  c: floor(3)
}
----

.Output
[source,JSON,linenums]
----
{
  "a": 1,
  "b": 2,
  "c": 3
}
----
