[[mod1]]
===== mod(Number, Number): Number

Returns the remainder after performing a division of the first number by the
second one.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  a: 3 mod 2,
  b: 4 mod 2,
  c: 2.2 mod 2
}
----

.Output
[source,JSON,linenums]
----
{
  "a": 1,
  "b": 0,
  "c": 0.2
}
----
