[[upper1]]
===== upper(String): String

Returns the provided string in uppercase characters.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
output application/json
---
{
  name: upper("mulesoft")
}
----

.Output
[source,JSON,linenums]
----
{
  "name": "MULESOFT"
}
----
