= singularize

//* <<singularize1>>


[[singularize1]]
===== singularize(String): String

Returns the provided string transformed into its singular form.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::core::Strings
output application/json
---
{
  a: singularize("boxes"),
  b: singularize("wives"),
  c: singularize("feet")
}
----

.Output
[source,json,linenums]
----
{
  "a": "box",
  "b": "wife",
  "c": "foot"
}
----
