= pluralize

//* <<pluralize1>>


[[pluralize1]]
===== pluralize(String): String

Returns the provided string transformed into its plural form.

.Transform
[source,DataWeave, linenums]
----
%dw 2.0
import * from dw::core::Strings
output application/json
---
{
  a: pluralize("box"),
  b: pluralize("wife"),
  c: pluralize("foot")
}
----

.Output
[source,json,linenums]
----
{
  "a": "boxes",
  "b": "wives",
  "c": "feet"
}
----
